## 项目概述

本项目是一款基于Python语言的民宿推荐系统，旨在为用户提供高效、准确的民宿预订服务。

## 技术栈

- 后端：Python Flask
- 数据库：MySQL
- 前端：HTML + CSS + JavaScript

## 功能模块

- 用户注册与登录
- 民宿搜索与筛选
- 民宿详情展示
- 用户评价与互动
- 推荐算法

## 系统角色

- 普通用户：浏览、搜索、预订民宿
- 管理员：管理民宿信息、用户信息、评论等

## 运行环境

- 操作系统：Linux或Windows
- Python版本：3.6+
- 数据库：MySQL 5.7+
